# Play2 Emu — Changelog

## **v1.0.0 — Initial Release**
### Released: 2025-01-01
- Initial Android + iOS builds
- Added support for:
  - `.iso`
  - `.chd`
  - `.bin/.cue`
- Touchscreen controls with L3/R3 gestures
- BIOS loader & verification
- Fully working PS2 boot sequence
- OpenGL & Vulkan rendering
- Audio output (SPU2)
- Game browser & cover downloader
- In-game pause menu
- Memory cards support
- Screenshot & video capture
- Patch codes system
- Disc changer
- Settings system with:
  - Interface
  - Graphics
  - Audio
  - OSD
  - Controller settings
- Hotkeys
- Transfer data system
- Screenshots & video capture manager
